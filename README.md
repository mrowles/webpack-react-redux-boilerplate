# Webpack React Redux Boilerplate

A small boilerplate of a [React](https://reactjs.org/) & [Redux](https://redux.js.org/) project being built by [webpack](https://webpack.js.org/).

## Dependencies

* Node

## Setup

`npm install`

## Build

`npm run build`

## Start

Open `index.html` in browser
